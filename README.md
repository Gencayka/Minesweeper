# Minesweeper
Версия JDK 15.0.2, для сборки проекта используется Gradle.

Minesweeper with MVC Воссоздание стандартной игры Windows XP Minesweeper с некоторыми упрощениями (нет звуков, монохромного режима, меток ?, меню Help содержит лишь кнопку About).

Игра реализована с применением паттерна MVC, представление с контроллером и модель с представлением общаются с помощью наблюдателей (обзерверов), контроллер вызывает методы модели напрямую.

Для реализации GUI использован Swing.

Данные о таблице рекордов содержаться в XML-файле в папке с игрой. Если на момент запроса к таблице хранящий ее файл не удается прочесть или содержащиеся в нем данные не соответствуют требуемым, создается новый XML-файл.

Все спрайты хранятся в папке \src\main\resources.

Некоторые параметры программы вынесены в файл data.properties в папке resources. Если какое-либо из значений, данных в файле не валидно, вместо него береться захардкоженное дефолтное значение. Данные из файла data.properties считываются и проверяются на валидность при запуске программы. Считанные (либо дефолтные, если значения не валидны или файл не удалось прочесть) значения записываются в поля объекта синглтон-класса Settings и далее считываются из него.
